# LEGACY

Seluruh isi direktori ini merupakan eksperimen, dataset kerja, skrip, dan artefak dari metodologi lama.

Materi ini **tidak digunakan pada skripsi final** dan tidak boleh dijadikan sumber angka, tabel, gambar, ground truth, atau kesimpulan penelitian. Sebagian artefak masih menggunakan Guideline-Based Labeling, tiga kelas `HIGH`/`MEDIUM`/`NORMAL`, Decision Tree C4.5, 105 atau 107 observasi, serta hasil eksperimen yang sudah tidak berlaku.

Eksperimen final menggunakan:

- `dataset_invoice.xlsx` sebagai sumber data;
- label historis admin sebagai ground truth;
- kelas `Urgent` dan `Not Urgent`;
- `run_comparative_experiment.py` sebagai skrip eksperimen; dan
- `comparative_experiment/` sebagai keluaran resmi.

Isi direktori legacy dipertahankan hanya untuk riwayat revisi dan keterlacakan proses penelitian.
